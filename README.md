# EdgeLab 🚀
> A curated learning hub for Embedded Linux, IoT, Networking, and AI/ML engineers.

🌐 **Live Site:** [abdelkarim13.github.io/EdgeLab](https://abdelkarim13.github.io/EdgeLab/)

---

## About

EdgeLab is a hand-picked resource platform designed for engineers building at the edge — from bare-metal Linux systems to intelligent IoT deployments. Every course and resource is carefully vetted and organized into focused learning tracks.

---

## Learning Tracks

| # | Track | Topics |
|---|-------|--------|
| 01 | **Embedded Linux** | Modern C++, Linux Fundamentals, Kernel Development, Device Drivers |
| 02 | **Internet of Things** | IoT Devices, Embedded Systems, IoT Security, Embedded ML |
| 03 | **Networking & Scripting** | Computer Networking, Bash Scripting, Python Networking, MySQL & SQLite |
| 04 | **AI / ML** | Intro to AI, Machine Learning, Deep Learning |
